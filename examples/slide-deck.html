htmd

Token-cheap structured data → rich self-contained HTML.

A library of templates for AI agents. v0.1 — May 2026

The problem

  • Agents writing HTML directly burns tokens (~2.5× MD).
  • Agents writing MD is cheap but visually flat.
  • We want rich output for the price of MD.

The idea

  1. Agent emits ~150 tokens of YAML.
  2. htmd render produces ~1,500 tokens of beautiful HTML.
  3. Charts, layout, theming, all on CPU. Zero LLM cost.

Killer demo

htmd render decision-matrix --data db.yaml

Same data, three forms:

  • MD: 1× tokens, flat table
  • Direct HTML: 2.5× tokens, decent
  • htmd template: 0.4× tokens, gorgeous heatmap

9 templates in v0.1

Tier B (interactive): slide-deck · prompt-tuner · kanban-board · concept-explainer

Try it

MIT licensed. Plugin SDK for community templates. → github.com/htmd/htmd

npm install -g htmd
htmd render dashboard --data metrics.yaml --out dash.html
htmd templates  # list all
